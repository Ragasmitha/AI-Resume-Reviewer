# AI Resume Reviewer
An AI-powered ATS Resume Reviewer web application that analyzes resumes and provides professional feedback, ATS compatibility insights, improvement suggestions, and resume scoring.

## Features
- ATS-Compatible Resume Analysis
- AI-Powered Resume Evaluation
- Target Job Role Selection
- Resume Score Generation
- Strengths & Weakness Detection
- Personalized Improvement Suggestions
- PDF Resume Upload Support
- Responsive and Modern UI
- Fast Real-Time Analysis
- Mobile-Friendly Design
