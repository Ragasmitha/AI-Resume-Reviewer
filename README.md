# AI Resume Reviewer
An AI-powered ATS Resume Reviewer web application that analyzes resumes and provides professional feedback, ATS compatibility insights, improvement suggestions, and resume scoring.

## Features
- Resume PDF Upload
- ATS Compatibility Analysis
- AI-Powered Resume Feedback
- Resume Score Generation
- Skill & Improvement Suggestions
- Modern Responsive UI
- Mobile Friendly Design

## Tech Stack
### Frontend
- React.js
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
- PDF-Parse
- Groq AI API
